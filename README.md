# To-Do List App ✅

A clean, fully functional task manager built with pure **HTML, CSS, and JavaScript**.  
Add tasks, edit them inline, mark as complete, and delete — all saved in your browser.

---

## 📁 Project Structure

```
todo-app/
├── index.html      # App layout and structure
├── style.css       # Dark theme styling
└── script.js       # All app logic (add, edit, delete, filter)
```

---

## ✨ Features

- ➕ **Add tasks** — type and press Enter or click Add
- ✏️ **Edit inline** — click the edit button to rename any task
- ✅ **Mark complete** — click the checkbox to toggle done/undone
- 🗑️ **Delete tasks** — remove any task instantly
- 🔍 **Filter tabs** — view All / Active / Completed tasks
- 💾 **Auto-save** — tasks are saved in `localStorage`, survive page refresh
- 📊 **Live stats** — shows total tasks and how many are done

---

## 🛠️ Tech Stack

| Technology  | Purpose                          |
|-------------|----------------------------------|
| HTML        | Page structure and layout        |
| CSS         | Dark theme, animations, styling  |
| JavaScript  | App logic, DOM manipulation      |
| localStorage| Saving tasks in the browser      |

---

## 🚀 How to Run

No installation needed. Just open the file in any browser.

```
1. Download or clone this repo
2. Open index.html in your browser
3. Start adding tasks!
```

Or visit: `file:///path-to-folder/index.html`

---

## 🧠 Concepts Used

- **DOM Manipulation** — dynamically creating, updating, and removing elements
- **Event Handling** — onclick, onblur, keydown events for interactivity
- **localStorage API** — persisting data in the browser without a backend
- **Array Methods** — filter, map, push for managing task state
- **contentEditable** — inline editing without extra input fields
- **CSS Animations** — smooth slide-in effect for new tasks
